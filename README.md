Проект «Метро»
Каждый поезд метро состоит из 5 вагонов. Три из которых могут быть любыми, а первый и последний только определенного типа (головные вагоны). В вашем распоряжении есть Депо с вагонами, сформируйте несколько поездов.
Поезд не должен быть готовым к поездке до тех пор, пока у него не будет пяти вагонов подходящего типа.
