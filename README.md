Проект «Метро»
Существуют вагоны двух типов: обчыные и головные. Каждый поезд метро состоит из 5 вагонов. Три из которых могут быть любыми, а первый и последний только головным. 
В депо (каждый вечер) загоняют все вагоны в случайном порядке. 
1. Наполните депо вагонами. 
2. Сформируйте несколько поездов из вагонов, принимая во внимание, что одновременно из депо может выехать один (последний) вагон. Далее этот вагон может быть помещен в любое место поезда. 
3. Сформируйте пул поездов, готовых выехать на линии (у  них есть один головной и один конечный вагон и вагонов всего 5)

Проект «Метро»
Киевское метро. 
1. Вагоны метро бывают двух типов: обычные и головные
2. Создайте депо с вагонами, набранными случаным образом.
3. Каждый поезд метро состоит из 5 вагонов. Три из которых могут быть любыми, а первый и последний только головными вагонами. Сформируйте несколько поездов, которые смогут ехать (у них есть один головной и один конечный вагон и вагонов всего 5). 
4. У вас не должно быть одинаковых поездов. Выпустите поезда на три линии метро (поочерёдно заполняя линии). 
5. Рассадите в каждый поезд машинистов, в зависисмости от их опыта (на первый подъехавший поезд садится самый опытный). После окончания рейса опыт машиниста увеличивается (для однопоточной программы – может и уменьшаться).
6. На каждой линии её поезда начинают проезд по станциям своей линии. С прибытием поезда, на каждой станции появляется несколько пассажиров.
7. На станции из кажого вагона прибывающего вагона выходит случайное число пассажиров. После этого в вагоны садится максимально возможное число пассажиров из числа находящихся на станции. В вагон может сесть ограниченное число пассажиров. Если мест больше нет, пассажиры ожидают следующий поезд. 
8. После проезда всех поездов выведите список пассажиров на конечных станциях всех линий.
